# React Router v6 to v7 Codemod

An automated, architecture-ready codemod for migrating React Router v6 codebases to v7, ensuring zero false-positives using `jssg` (JS ast-grep).

## Architecture

This project was built using a specialized team of autonomous agents:
- **Architect**: Designs the sequences in `workflow.yaml`
- **jssg Transforms Engineer**: Implements exact AST replacements in `jssg/`
- **Test Engineer**: Ensures quality via `tests/`
- **Documentation Writer**: Compiles this registry listing
